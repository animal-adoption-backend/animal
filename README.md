## 4. 핵심기능

+ **회원가입, 로그인**   
  : JWT를 이용하여 로그인과 회원가입을 구현하였습니다.
  : bcrypt를 이용하여 암호화된 비밀번호를 DB에 저장합니다.
  
+ **동물 등록**   
  : 로그인 후 유기동물의 이름, 사진, 설명을 등록합니다.
  
+ **댓글**   
  : 등록된 동물에  댓글을 작성할 수 있습니다.
  : 자신이 작성한 댓글은 로그인 후 수정 및 삭제 할 수 있습니다.
